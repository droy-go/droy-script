# Basic Example Package

A simple example demonstrating the basic structure of a Droy package.

## Files

- `droy.toml` - Package configuration
- `src/main.droy` - Main source file
- `README.md` - This file

## Usage

```bash
# Install dependencies
droy-pm install

# Run the package
droy run src/main.droy
```

## Output

```
Hello from Basic Example!
Sum of 10 and 20 is: 30
```
